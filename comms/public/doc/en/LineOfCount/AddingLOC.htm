Assigning Line of Count information

You can assign Line of Count (LOC) information to cables or to the port(s) of a piece of equipment.

Assigning LOC to a cable or an equipment item

To assign LOC:

  1. Select the structure that contains the equipment. You can enter (part of) of its name in the Search box and select it from the list of hits or you can locate the object on the map and select it there.

  2. In the Equipment tree, expand the equipment to which you would like to assign LOC information. connect the cable.

    The incoming (IN) and outgoing (OUT) ports for the equipment are listed.

    OR

    In the Cable tree, open the node for the required cable.

    The Edit button is only available if the selected feature has no LOC assigned to it yet or if it is the origin of LOC information.

  3. Select the IN or OUT port to which you want to assign the LOC.

  4. Right-click the selected port, and click Edit LOC from the context menu.

    The Edit LOC dialog appears.

  5. Enter the data to compose the LOC

  1. Click Add to create a record.
  2. Enter the Name that you want to give to the LOC record.
  3. Enter the values for the Low and High bounds of the logical range of strands. 

    The values for the physical range are updated automatically.

  4. Select the Status: the choice is between Active, Spare or Dead.
  5. If you only want the status part of the LOC displayed, you can check the Status Only box

  1. Repeat the above step if you want to add another record.

  2. If needed you can change the order of the LOC records by clicking the icon at the beginning of a record and dragging it into the desired position.

  3. Click Ripple if you have completed all the information to pass the LOC information down through all physically connected strands.

  4. Add this stage you can save the results if you are satisfied with them or you can cancel them and reiterate to make the adjustments needed.

    If you click Cancel the Edit LOC dialog is closed. If there are unsaved changes, a confirmation dialog is shown first.

Rippling the LOC information

When you have assigned the LOC information to the port of an equipment item or to a cable or have changed the data, you have to use the Ripple button to pass it from the origin segment down through physically connected strands and have the LOC applied to those strands.

The LOC information is assigned to the cable segments and visible as Line of Count Sections.

Other template models may be available for your application domain. Contact IQGeo support for a list of available modules.

 

Template data models are installed using the comms_db command-line tool. These are the steps to do so:

  1. Go to the tools directory in your installation (or add it to your PATH):

    cd <myWorld>/Webapps/myworldapp/modules/comms/tools

  2. Install the model:

    comms_db <db_name> install <model_name>

Once the model(s) installed, you can modify and extend the objects created according to your own preferences and requirements. To do this, go to the configuration pages Features and Settings > Comms in the Configuration app.

Adding a polygon

The Markup Polygon item enables you to add polygons to the design and update them.

You can either add a new polygon and then add it to a referenced feature in the design, or you can select an existing referenced feature first and then add a new line to it.

Adding a new polygon and associating it with a referenced feature

  1. Click the Markup Polygon item on the Markup Palette. The Markup Polygon details are displayed in the Details pane.

Figure: Markup Polygon details

  1. Position the cursor over the place on the map where you want to start drawing a polygon.

  2. Click the left mouse button to anchor the cursor to the starting point of the drawing.

  3. Drag the cursor across the map until you reach the position you want the first stage of drawing the polygon to stop.

  4. Click the mouse button to fix the first stage and then continue dragging until you reach the position of the next stage.

  5. Continue to click and drag until you have finished drawing the polygon.

  6. Double click the cursor to stop drawing and exit drawing mode.

You can click and drag the polygon to a different location on the map if required.

You can now associate the polygon with a referenced feature in the design.

  1. Click the Edit Feature icon beside Referenced Feature to open the following dialog.

Figure: Set 'Referenced Feature' dialog

  1. Click the feature in the design to which you wish to reference the polygon. It is added to an incremental list inside the dialog.

  2. Click the OK button to save the referenced feature and go back to the Details pane.

You can now add the fill color, dash style, outline color and the outline width (in pixels or meters).

If you click the Cancel button on the Details pane, edit mode is exited and the Markup Polygon details on the Details pane are closed.

The polygon is saved to the design as shown on the Details pane below.

Figure: Markup Polygon saved to the design

The polygon remains displayed on the map. You can either click on it to reopen the Markup Polygon details on the Details pane and edit it, or delete it.

The polygon is automatically saved once you have drawn it on the map. Clicking the Save button on the Details pane confirms the save and closes the Markup Polygon details on the Details pane. The polygon is saved to the design as shown on the screen shot above.

 

Figure: Markup Polygon added

Adding a new polygon to an existing feature

  1. Click the feature in the design to which you wish to add a polygon.

The Markup Polygon details are then shown in the Details pane (shown in the Markup Polygon details screen shot).

  1. Click the Markup Polygon tool and start to draw the polygon. The process is as described in Adding a new polygon and associating it with a referenced feature.

You will need to define one or more layers to display your design features types. If desired, you can use IQGeo layer filtering to control which users can see which designs.

It is also possible to define forward view layers that display the unpublished data from designs on the map view. This provides an overview of the planned work for an area.

To define a forward view layer, include a schema parameter in your layer definition.

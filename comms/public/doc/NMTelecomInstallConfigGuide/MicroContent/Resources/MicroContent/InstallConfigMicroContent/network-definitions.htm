Network Manager adds the following network definitions to the system.

Name Description
mywcom_cable_segments Mesh network for tracing structures and fiber cable segments.
mywcom_coax Tree network for modeling coaxial wires and equipment.
mywcom_copper Tree network for modeling copper pairs and equipment.
mywcom_fiber Tree network for modeling fiber equipment and cables.
mywcom_fiber_path Tree network for finding multiple candidate paths in fiber network.
mywcom_routes Mesh network for modeling structures and routes.
These definitions are critical to the operation of Network Manager tracing. They can be modified but must not be deleted.

The State Manager component controls the design states in which a user is allowed to modify data. Once enabled, users can change data only when:

  • They are in a design and
  • That design is in an editable state.

The State Manager is configured using the Advanced tab of the Settings configuration page. The relevant settings are listed in the following table.

Setting Description
mywcom.stateManager Enables/disables State Manager.
mwycom.editableStates Lists the design states in which data is editable.

The Network Manager (Comms) product provides:

  • An application for creating and editing telecoms network data including structures, equipment, routes, conduits and cables.
  • Data model for modeling structures, equipment, cables, specifications and fiber color schemes.
  • Layer definitions, network definitions, and JavaScript models for network object types.
  • JavaScript models and Python services to support the application.
  • A custom configuration page for editing settings that are specific to Network Manager.
  • Facilities for integrating with IQGeo Platform versioning (design management).
  • Special rights for configuring the Network Manager functionality.

The module also provides includes an optional template data model that provides a quick start for configuring your system.

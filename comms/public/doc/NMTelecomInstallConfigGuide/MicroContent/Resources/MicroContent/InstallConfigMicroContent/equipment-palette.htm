To configure the default Equipment palette:

  1. Log in as the default user, and launch the Comms application.

  2. In the toolbar, click the Equipment Palette icon .

    Note that the palette will be empty when you open it for the first time.

  3. Add the required objects to the palette, as described in the User Guide.
  4. Click the Save Application State button on the Layers tab to save your palette configuration to the database.

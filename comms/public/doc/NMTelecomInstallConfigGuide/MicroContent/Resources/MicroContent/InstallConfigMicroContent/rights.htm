The following rights to the system are provided:

Role Allows the user to...
Comms: Bulk-move features Move multiple objects to a different location on the map in the main application.
Comms: Edit master

Make changes directly to the master view of the database.

Users who have this right will also be able to open and edit a design regardless of the design group assignment.
Comms: Manage Labor Costs Add and modify labor cost records.
Comms: Manage specifications Add and modify specification records.
Comms: Publish Designs Access to the Publish icon in the Design toolbar.
Comms: Run data validation Access the Data Validation tool.
Comms: View All Designs

Users who have this role will be able to view all designs but do not have Admin rights and do not belong to any group.
Edit User Groups

Access to the User Groups icon to open, edit and manage user groups.
Modify layer list Access to the functions to add layers the list, change their order and arrange their rendering order on the map.
Save current state to database Users who have this right are allowed to save the application settings and layers that are visible, the map view and various bits of information about the state of the UI.
Save default state for all users Users who have this right are allowed to save the application state to make it available to all users.

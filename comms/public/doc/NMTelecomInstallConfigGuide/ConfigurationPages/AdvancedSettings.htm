Advanced Settings

Besides the Network Manager allows you to configure other areas of the system. To access these settings, go to the Advanced tab of the Settings Configuration page.

Setting Description Example
iqgapp.markup.featureTypes

The list of markup features and their default styles is controlled by this setting.

This list is also used to derive the default markup palette.

Please ensure that you do not modify these settings without guidance.

mywcom.bulkMove

Defines whether bulk move (selecting and moving multiple objects to a different location on the map) is enabled or disabled.

Defaults to a maximum of 30 objects that can be moved at the same time but this can be changed as required.

 
{
 "enabled": true,
  "selectionLimit": {
    "max": 30,
    "min": 1
  }
}

mywcom.pathfinder

These settings are used to control the behavior of Path Finder and the creation of splice closures when it creates new connections for circuit routing.

The splice_closure_type setting must be set to the name of the feature in the data model that will contain fiber connections, automatically created when routing a circuit along a path. The splice_closure_properties can be populated with default values for splice closure features created.

For example:

"splice_closure_type": "splice_closure",

"splice_closure_properties": {

"specification": "CS-FOSC400-A4-16-1-BNV"

},

The other settings can be modified to tune the Path Finder for particular networks.

For details, contact IQGeo Support.

 
{
  "max_nodes": 1000,
  "show_partial": false,
  "max_route_paths": 10,
  "max_route_nodes": 1000,
  "exclude_similar": false,
  "similarity_threshold": 90,
  "route_paths_only": false,
  "cable_paths_only": false,
  "bidirectional": false,
  "num_back_steps": 20,
  "max_paths_default": 5,
  "max_dist_default": 5000,
  "connection_cost_metres": 10,
  "splice_closure_type": "",
  "splice_closure_properties": {
    "specification": "",
  },
  "include_as_via": true
}
mywcom.strandlayout

The Layout Strand setting allows system administrators to designate which feature types participate in the tool.

It defines if the tool is enabled in the system. When in placement mode, the length of the current line segment is shown on the map in feet. The setting allows you to change the length to meters.

It also designates which structures and route types are overhead vs underground and which fields they can populate. 

{
  "defaultLengthUnit": "ft",
  "enabled": true,    
  "overhead": {
    "structures": {
      "pole": {
        "fields": [
          "height",
          "type"
        ]
      },
      "building": {
        "fields": [
          "owner",
          "access_code"
        ]
      }
      "routes": [
        "oh_route"
      ]
   },
  "underground": {
    "structures": {
      "manhole": {
	 "fields": [
	   "size_x",
	   "size_y",
	   "size_z"
	 ]
      }
    },
    "routes": [
      "ug_route"
    ]
  }  
}
mywcom.systemChangesLimit

The Show System Changes UI has an input to limit the number of results.

A configurable, maximum limit can be set so that the system is not overwhelmed.

The configured maximum limit value is honored, regardless of the input value.

1000

Network Manager objects

Types of object

Network Manager models the following types of objects.

Category Containment Description Examples
Structure Top level Support structure, chamber, or building. Building, cabinet, manhole, pole
Route Top level Path between two structures. Trench, aerial route
Equipment Contained

Signal transmission equipment or equipment enclosure. Can be nested.

The range of network equipment items

Splitter, splice closure, rack, slot
Conduit Contained Duct within a route. Can be nested. Duct
Cable Contained Cable passing through routes/ducts. Connects to equipment at multiple points along its path. Fiber cable
Circuit Contained Physical path of a service. Customer circuit

Top level objects have geometry and are visible on the map. Other categories also have geometry, but it is derived from their container (housing).

The specific object types available to you, their attributes, and containment rules, are defined by your System Administrator. For information about configuring network objects, see the Installation and Configuration Guide.

Adding objects to a design

With the Add Object tool, Network Manager provides an easy and flexible way to add any of the configured network object types to your design.

To do so, click the Add Object icon on the Network Manager toolbar.

The Add Object dialog appears, presenting an overview of the available object types grouped per type. The examples shown below are taken from an application that has the data model templates installed for fiber, coax and copper. The list therefore includes equipment items and cables for all three technologies.

Figure: Add Object dialog

From that list, select the object type that you want to add.

Containing structure determines available equipment

The Equipment section is only populated if a containing structure is selected. Moreover, only equipment types compatible with the selected structure are displayed.

The three examples below, illustrate respectively, a situation with no structure selected, with a manhole selected and with a pole selected.

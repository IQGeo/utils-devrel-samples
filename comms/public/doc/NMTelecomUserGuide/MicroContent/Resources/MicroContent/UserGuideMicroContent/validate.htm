To check an area of data:

  1. Zoom to the area on the map that contains the objects that you would like to check.

On the toolbar:

  1. Click the Tools palette icon. The palette appears.

Figure:  

In the Tools palette:

  1. Click the Data Validation icon. The Data Validation dialog appears.

Figure: Data validation

  1. Select the objects that you would like to check.

  2. Select Start.

The application performs the specified checks. If any issues are found, these are listed in the Details tab.

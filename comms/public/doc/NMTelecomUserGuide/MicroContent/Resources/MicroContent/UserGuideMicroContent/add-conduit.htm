To add a conduit:

  1. On the toolbar, click the Add Object icon , and then select the type of conduit to add, for example Blown Fiber Tube.

  2. On the map, draw a line between the required start and end structures.

    Figure: Adding a conduit

  3. In the Details tab, set the attributes for the conduit. Note that if your administrator has configured specifications for the object type, you can select multiple attributes, as described in Setting a specification.

  4. Save your changes. The conduit is added. Note that if your conduit object has been configured with one, you can place multiple conduits along the path by entering a value in the count field.

    Figure: Example conduit

Notes:

  • A separate conduit object is created in each route along the structure. In the above example, the application creates two conduits:

    • Conduit XX-CND-2004, between XX-M-117 and XX-M-118.
    • Conduit XX-CND-2003, between XX-M-118 and XX-M-114.
  • The Details tab lists all the conduits that are created. When you hover over the details, the relevant conduit is highlighted on the map.

  • Depending on the conduit type you selected, the conduits may be open at each structure or continuous (passthrough).

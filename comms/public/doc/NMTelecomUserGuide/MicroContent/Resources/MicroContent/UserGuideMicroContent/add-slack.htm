To add a cable slack:

  1. On the map, select the structure that will house the slack. You can either use the Search box or select the object on the map.

  2. In the Cable tree, open the node for the required cable.

  3. Right-click the IN or OUT node, and then click Add Slack.

  4. In the Details tab, set slack length and any other attributes.

  5. Click Save.

  6. Reselect the structure. The Cable tree shows slack.

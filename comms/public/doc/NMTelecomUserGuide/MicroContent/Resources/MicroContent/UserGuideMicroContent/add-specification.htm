For some object types, your Administrator may have configured a specification catalog. Specifications provide a quick way of setting multiple attributes of an object. They also provide metadata (manufacturer, cost, and so on) that is used in generating bill of materials reports.

Object types that have a specification catalog will have a Spec field in their editor.

To set a specification:

  1. Add the object.

    The Details tab displays a form, where you can set attributes.

  2. Click the Spec field. The Spec Catalog appears, listing the available specifications.

    Figure: Specifications

  3. If you have already specified some attributes for your object, only the specifications that match those attributes are displayed.

    Figure: Matching specifications

  4. In the catalog, click the required specification. The attributes on the editor are populated.
  5. Save your changes.

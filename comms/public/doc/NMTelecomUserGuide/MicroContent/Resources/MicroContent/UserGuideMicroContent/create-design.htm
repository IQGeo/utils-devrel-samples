To create a design:

  1. Pan and zoom to the area of the map, where you would like to create your design.

  2. On the toolbar, click the Add object icon , and then select the type of design that you would like to create, for example Customer Connection Job.

  3. On the map, draw the boundary of your design.

Figure: Adding a design

  1. In the Details tab, set the attributes for the design.

The Group drop down list allows group owners and design owners to restrict who can access a design by assigning a design to a user group. This enables you to track changes that users make to designs.

Figure: Group drop down list

The design is now restricted to the assigned group and the group owner. Members of any other group will not be able to view it.

  1. Click Save. The design is created. The Details tab displays the Design toolbar, which includes a set of icons that allow you to manage your design.

Figure: Design toolbar

The design object is created. For information about adding data to the design, see Opening a design.

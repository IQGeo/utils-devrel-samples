Adding a structure

A structure can be one of the following:

  • Building

  • Manhole

  • Cabinet

  • Pole

  • Drop Point

  • Wall Box

  • MDU Multi Dwelling Unit, such as an apartment building.

 

To add a structure:

  1. Pan and zoom to the area on the map, where you would like to add the structure.

  2. On the toolbar, click the Add Object icon .

  3. Select the required object type, for example pole.

  4. Click the point on the map where you would like to place the object.

  5. Set the required attributes for the object.

  6. Click Save. The object appears on the map.

    Notes:

    • For some types of objects, you may be able to set properties from a catalog of specifications that have been configured by your Administrator. For more information, see Setting a specification.
    • When you save, some attributes, for example name, may be set automatically.
    • Some attributes may be mandatory. The feature will not be saved until these attributes are set.

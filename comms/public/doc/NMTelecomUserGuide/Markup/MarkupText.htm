Adding a text box

The Text markup object enables you to add a text box to the design.

You can then add text, click and drag the text box to move it to a different location, resize it and rotate it, as required.

You can either add a new text box and then add it to a referenced feature in the design, or you can select an existing referenced feature first and then add a new text box to it.

Adding a new text box and associating it with a referenced feature

  1. Click the Markup Text item on the Markup Palette.The Markup Text details are displayed in the Details pane.

Figure: Markup Text details

  1. Position the cursor over the place on the map where you want to start drawing the text box.

  2. Click the left mouse button to drop the text box onto the map.

You can click and drag the text box to a different location on the map if required.

You can start typing the text required to be displayed in the text box, in the Text field on the Details pane.

You can now associate the text box with a referenced feature in the design.

  1. Click the Edit Feature icon beside Referenced Feature to open the following dialog.

Figure: Set 'Referenced Feature' dialog

  1. Click the feature in the design to which you wish to reference the text box. It is added to an incremental list inside the dialog.

  2. Click the OK button to save the referenced feature and go back to the Details pane.

  3. You can also now specify whether there will be a leaderline by enabling the check box beside the Leaderline field. You can add the color of the text box and the text size in pixels, the text box background color and the outline width in pixels.

Figure: Markup Text Dialog - Text and attributes added

If you have specified a leaderline, the start of the line is the position of the text box.

  1. Click and drag the text box from the starting position, you will see that a leaderline is linked to the text box.

  2. Release the mouse button to anchor the text box and leaderline.

The text box can be moved, resized and rotated, and you can write inside it as shown in the following diagram.

Figure: Manipulating the text box

If you click the Cancel button on the Details pane, edit mode is exited and the Markup Text details on the Details pane are closed.

The text box is saved to the design as shown on the Details pane below.

Figure: Markup text box saved to the design

The text box remains displayed on the map. You can either click on it to reopen the Markup Text details on the Details pane and edit it, or delete it.

The text box is automatically saved once you have drawn it on the map. Clicking the Save button on the Details pane confirms the save and closes the Markup Text details on the Details pane. The text box is saved to the design as shown on the screen shot above.

Figure: Markup text added

Adding a new text box to an existing referenced feature

  1. Click the feature in the design to which you wish to add a text box.

The Markup Text details are then shown in the Details pane (shown in the Markup Text details screen shot).

  1. Click the Markup Text tool and start to draw the text box. The process is as described in Adding a new text box and associating it with a referenced feature.

Editing address details and modeling the end of the network

This section provides an example of how you can model the customer end of an FTTH Fiber To The Home. network, making use of an address feature and a reference on the address, back to the equipment that serve the address.

Your system administrator will need to add the address feature to the system if you wish to make use of this example.

The following is an example of how you can search for an address as well as view and edit the relationship it has to the equipment

Searching for an address

Firstly:

  1. Search for the address in the Search bar.

Then:

  1. Select the address so that it is displayed both in the Details pane and on the map.

Figure: Address details

  1. Click the Edit icon to make the address fields editable.

You can now see, on the map, that the number of the address is indicated by a label, displayed in large digits (see example screen shot below).

Figure: Editable address details

Editing an address

You can amend the following address data items, if required:

  • Name, Number, Street, City, Postcode and Service Status.

The Serving Structure data item enables you to select an object on the map, to be the serving structure associated with the address.

  1. Click the Edit icon beside the entry to display the following dialog:

Figure: Set 'Serving Structure' dialog

  1. Click the object required on the map. Its name is then displayed on the dialog, as shown in the screen shot above.

  2. Click the Done button to select the object. It is then shown in the Serving Structure data item in the Details pane.

The Serving Equipment data item is used for modeling the end of the network.

  1. Click the Save button to save the changes.

Modeling the end of the network

The address details area enables you to model the end of the network at your site, as follows:

  • Equipment sits inside structures on the map and can be selected via the address of the structure where it is located.

  • The Serving Equipment data item relates the address to the equipment or drop, that is serving it.

  1. Click the Edit icon. The following dialog is then displayed:

Figure: Equipment Reference Editor dialog

An equipment tree listing the equipment that serves the address, is shown in the dialog.

  1. Click the piece of equipment on the list, that you wish to select, to be the serving equipment at the address.

  2. Click the Done button to select the equipment. It is then shown beside the Serving Equipment item in the Details pane.

Figure: Serving Equipment data item with serving equipment selected

  1. Click the Save button to save the changes.

You can then navigate to the serving equipment by clicking the hyperlinked equipment name beside the Serving Equipment data item:

Figure: Serving Equipment - hyperlinked Serving Equipment name

The serving equipment details are then shown. There is a reverse reference set calculated, indicated by the Items hyperlink, beside Served Customers, as shown in the screen shot below:

Figure: Equipment details - Items reference set

  1. Click the Items hyperlink to open a list of customer addresses associated with the serving equipment:

Figure: List of customer addresses associated with the equipment

You can then carry out a bulk update with the addresses on that equipment, and set the status to be In Service. Please refer to the IQGeo Platform User Guide for details on how to do this.

REST API

Network Manager provides the following REST services for creating and updating feature data.

Routing Request Type Purpose
/modules/comms/feature POST Insert / update / delete a set of features, firing Network Manager server-side triggers.
/modules/comms/feature/{feature_type} POST Insert a feature firing Network Manager server-side triggers.
/modules/comms/feature/{feature_type}/{id} POST Update or delete a feature firing Network Manager server-side triggers.

These services are identical to the corresponding IQGeo Platform feature services except that they fire Network Manager Server-side Triggers.

Network Manager also implements a number of other REST services. Those services are used internally by the mywcom application. They are not guaranteed to be stable over releases.

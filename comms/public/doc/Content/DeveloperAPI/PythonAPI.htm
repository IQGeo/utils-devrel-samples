Python API

Network Manager provides a rich set of classes for manipulating network data in Python.

Class Purpose
NetworkView Broker for accessing managers.
StructureManager Maintains network consistency after structures and routes are inserted / updated / deleted.
EquipmentManager Handles moving and copying of equipment assemblies.
ConduitManager Manages conduit routing, connection, and disconnection.
CableManager Manages cable routing, addition of slacks, and so on.
ConnectionManager Manages fiber-level connection and disconnection.
CircuitManager Routes and unroutes circuits.

Note that these APIs may be subject to change at future releases.

The Python API can be accessed using the myw_db run command.

JavaScript API

The Network Manager JavaScript client provides a number of plugins for querying and manipulating network data.

Plugin ID Purpose
structureManager Find structures at point, trace through route network.
equipmentManager Move and copy equipment.
conduitManager Move, cut, and join conduits.
cableManager Add and remove slacks.
connectionManager Run fiber trace, connect and disconnect pins, query pin state.
circuitManager Route and unroute circuits.
displayManager Get fiber colors, build pin tree labels.
specManager Client-side cache for spec records.

Note that these APIs may be subject to change at future releases.

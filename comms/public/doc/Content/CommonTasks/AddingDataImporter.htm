Adding a Data Importer

A data importer is a server-side class that reads a data package and builds network objects from it. The product provides a data importer for IQGeo Comms Data Interchange Format (CDIF format). You can add additional importers for other package formats if desired.

To add a data importer:

  1. Create your data importer class (see below)

  2. Register it as an import engine by calling DataImporter.registerEngine()

A data importer provides the following methods:

Method Purpose
__init__(self,db_view,feature_pkg,...) Instantiates the class. Does no major work. feature_pkg is a FeaturePackage object (wrapper to a set of data files or similar).
detachedFeatures() Returns features from self.feature_pkg for display in GUI as preview of data load.
run() Reads data from self.feature_pkg and creates network objects from it. Superclass provides a number of helper methods for mapping IDs etc.

See cdif_data_import.py in comms/server/data_import for an example.

Introduction

Network Manager is a browser-based application for modeling telecoms networks inside and outside plant. It is implemented as a module within the IQGeo platform.

This document provides information for developers building custom functionality on top of Network Manager or migrating data into it.

Intended Audience

This guide is intended for developers who have a good understanding of the IQGeo platform and are familiar with telecoms terminology.

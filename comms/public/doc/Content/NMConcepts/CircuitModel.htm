Circuit Model

Circuits are modeled as a set of qualified reference_sets stored on mywcom_fiber_segments and equipment.

  • Circuit: Models a customer service. Inherits its geometry from participating mywcom_fiber_segment records.

  • mywcom_fiber_segment:Has field circuits which stores the relationship with circuits as a set of fibers in a given cable segment as qualified urns. Direction may differ.

    • ftth_circuit/1324?fibers=1

    • ftth_circuit/1426?fibers=1

    • bb_circuit/12?fibers=17:18

  • Equipment:Objects registered as equipment have a field circuits which stores the relationship between a circuit and a set of ports as qualified urns

    • ftth_circuit/1324?ports=in=1:1&out=1:1

    • ftth_circuit/1426?ports=in=1:2&out=1:2

    • bb_circuit/12?ports=in=11:12&out=11:12

For more details, see objects circuit in Data Model (System Objects).

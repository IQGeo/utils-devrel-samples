mycom_circuit_port

Field Type Comments
id integer Generated ID.
circuit reference Owning circuit.
root_housing reference Structure in which port(s) are located.
equipment reference Equipment on which port(s) are located.
side string(16) Side of equipment ('in' or 'out').
low integer Start of port range.
high integer End of port range.
in_segment reference Previous circuit_segment or circuit_port in chain (if any).
out_segment reference Next circuit_segment or circuit_port in chain (if any).
location point Location in geographic world. Must match location of root_housing.

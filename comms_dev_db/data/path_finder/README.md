A collection of CDIF packages for testing particular aspects of path finder.

-   GridNetwork3x3.zip - A 3x3 grid of routes along which cables run. Bottom left is the from-cabinet and top right is the to-cabinet.
-   GridNetwork5x5.zip - A 5x5 grid.
-   GridNetwork10x10.zip - A 10x10 grid.
-   InvalidSplices.zip - Testing exclusion of invalid splicing
-   MultiplePaths.zip - Testing multiple paths
-   MiniFTTH.zip - Cabinet with 48 fibre cable going north, another going east to end manhole and then drop cables from manhole to wallboxes.

Code to build the grid network files is in ./utils/create_pf_grid.py
